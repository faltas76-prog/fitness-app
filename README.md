# 🏋️ Fitness Planner App

Moderní fitness aplikace vytvořená pomocí HTML, CSS a JavaScriptu.

## 🚀 Funkce

- 📅 Týdenní tréninkový plán (accordion / roletky)
- ✅ Odškrtávání cviků
- 🔓 Postupné odemykání dní
- 📊 Progres (%) a streak systém
- ⚖️ Tracking váhy + graf
- ⏱️ Timer mezi sériemi
- 🔔 Notifikace
- 📱 Instalace jako mobilní appka (PWA)
- 🤖 AI trenér (OpenAI API)
